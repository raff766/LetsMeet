# Let's Meet

An app for meeting new people

## How to use?

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project on iOS / Android / Expo.

```js
yarn ios // yarn android // yarn start
// or
npm run ios // npm run android // npm start
```
